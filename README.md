# 🚀 AI SEO Blog & Content Cluster Generator (Task 3)

## 📌 Project Overview
This project demonstrates how AI + SEO + Prompt Engineering can be used to create a **complete content marketing system** for a business website.

Instead of writing random blogs, this project follows a **Content Cluster Strategy** used by professional SEO agencies.

---

## 🏪 Business Chosen
- **Business Type:** Dental Clinic  
- **Location:** Bangalore  
- **Target Audience:** Families, working professionals  
- **Goal:** Generate leads, build trust, rank on Google  

---

## 🎯 Objective
To build a system that can:
- Generate SEO blog outlines (H1–H3)
- Create long-form blog content
- Build content clusters (pillar + supporting blogs)
- Optimize content for local SEO
- Support business conversions (leads & bookings)

---
